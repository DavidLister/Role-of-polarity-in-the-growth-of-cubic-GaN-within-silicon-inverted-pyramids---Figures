# Role of polarity in the growth of cubic GaN within silicon inverted pyramids

This repository contains the raw image data and figures for the manuscript: *Role of polarity in the growth of cubic GaN within silicon inverted pyramids*. Currently available as a preprint on [arXiv:2606.30924](https://arxiv.org/abs/2606.30924) and submitted to *Applied Physics Letters*.

## Contents
- Each folder corresponds to a figure and contains all the requisite source files.

## Methodological notes
- Figures were prepared using Scribus and have small brightness and contrast adjustments for clarity. 
- The atomic model was made using [MSEP.one](https://github.com/MSEP-one/msep.one) software.
